# notre Projet Web Touristique du Maroc

## 🌍 Présentation du Projet

**Morocco Tourism Web Project** est un site web touristique moderne, responsive et immersif conçu pour mettre en valeur la beauté, la culture et la diversité du Maroc.
La plateforme présente les destinations emblématiques, la gastronomie marocaine, les activités touristiques ainsi que les expériences uniques du Royaume à travers une interface élégante et interactive.

Ce projet combine des technologies web modernes avec une identité visuelle premium inspirée du patrimoine marocain, afin d’offrir aux utilisateurs une expérience numérique immersive à travers les différentes régions du Maroc.

📚 Ce projet a été réalisé dans le cadre du module de **Développement Web** à l’École Nationale des Sciences Appliquées de Tanger (ENSA Tanger).

# 🇲🇦 À Propos du Maroc

Le Maroc est une terre de diversité où histoire, culture et modernité coexistent harmonieusement.
Des montagnes de l’Atlas aux dunes du Sahara, en passant par les ruelles bleues de Chefchaouen et les souks animés de Marrakech, le Maroc offre des expériences inoubliables aux visiteurs du monde entier.

Ce projet a été conçu afin de promouvoir le potentiel touristique du Maroc à travers une expérience numérique moderne et immersive.
